# Clerk Notebooks

## Getting Started

Install clojure: https://clojure.org/guides/install_clojure#_linux_instructions

Read https://github.com/nextjournal/clerk#-using-clerk

Run with:

```
clj -M:repl/conjure
```

Go to http://localhost:7777/
